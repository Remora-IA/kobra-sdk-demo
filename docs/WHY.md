# WHY — kobra-sdk-demo

Este repo existe para que el primer desarrollador que integre Kobra (Somos Rentable u otro cliente)
pueda ir de cero a integración funcionando en menos de una hora, sin fricciones.

La métrica única: **tiempo hasta primer webhook recibido**.

Si el README miente, si un escenario no corre, si la API cambió y los scripts no — fallamos antes de que el cliente empiece.
